# XAI Digital Persona UI

본 프로젝트는 **Streamlit 기반의 금융 자산 관리 UI 프로토타입**입니다.  
AI 분석 결과를 사용자가 직관적으로 이해할 수 있도록  
자산 날씨, 변수 영향도, 포트폴리오 리밸런싱, 미래 시뮬레이션 화면을 제공합니다.

설명 가능한 AI XAI 개념을 금융 자산 관리 서비스 UI에 적용한 데모 목적의 프로젝트입니다.

---

## 주요 기능

### 1. 자산 날씨 및 시장 국면 요약
- 현재 시장 환경을 날씨 메타포로 시각화
- 사용자의 자산 상태에 대한 AI 종합 판단 제공

### 2. AI 분석 변수 설명
- 금리 환율 지정학적 리스크 등 주요 변수 시각화
- SHAP 지수 형태의 영향도 그래프 제공
- AI 판단 근거를 사용자에게 설명

### 3. 맞춤형 포트폴리오 리밸런싱
- 리밸런싱 전후 자산 비중 비교
- 파이 차트를 활용한 직관적인 변화 표현
- 연계 금융 상품 버튼 제공

### 4. 미래 자산 시뮬레이션
- 장기 자산 성장 곡선 시각화
- AI 리밸런싱 적용 시와 미적용 시 비교
- 은퇴 자산 목표 달성 가능성 안내
